G. COMMON INTEGRATION ISSUES & SOLUTIONS
===================================================================================

Issue 1: "Undefined control sequence" errors
──────────────────────────────────────────
  Cause: LaTeX commands in content not defined
  Solution: Ensure preamble includes all required packages
  Check: \usepackage{amsmath,amssymb} for mathematical symbols

Issue 2: Bibliography citations show as [?]
──────────────────────────────────────────
  Cause: Citation keys don't match references.bib
  Solution: Run full compilation cycle: pdflatex → bibtex → pdflatex → pdflatex
  Check: grep "cite{" sections/*.tex matches entries in references.bib

Issue 3: Figures don't render
────────────────────────────
  Cause: Image files referenced but not present
  Solution: Ensure Pictures/ folder exists with required images
  Check: \includegraphics[width=\linewidth]{Pictures/filename}

Issue 4: Section numbering jumps (1.1 → 2.5)
───────────────────────────────────────────
  Cause: Nested \subsection commands or missing \section breaks
  Solution: Verify correct nesting: \section → \subsection → \subsubsection
  Check: Use grep "^\\section\|^\\subsection" to verify structure

Issue 5: References to Section "??" appear
────────────────────────────────────────
  Cause: \label or \ref commands mismatched
  Solution: Ensure labels are defined before reference
  Check: \label{sec:methods} defined before Section mentioned as \ref{sec:methods}
